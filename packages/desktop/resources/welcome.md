# Welcome to NoteCove

Your notes, **beautifully organized** and _always in sync_.

## Getting Started

Here are some things you can try:

- Open multiple windows to see **real-time collaboration**
- Create folders to organize your notes
- Use `Cmd+N` to create a new note

## Formatting Examples
